# Analysis on job postings made in Mexico City

**Code based on David Sheehan's publication. Link to original post [here](https://dashee87.github.io/data%20science/data-scientists-vs-data-analysts-part-1/)**

The present repository store the source code for the analysis made in R. A RData that contains the data set used in the analysis and a markdown file to present the results.

The analysis was published (in Spanish) on R Pubs: [Link](http://rpubs.com/carian2996/oferta-laboral-actuario-mexico)
